# Static Timing Analysis (STA) using SDC

## Overview
This project demonstrates Static Timing Analysis (STA) using Synopsys Design Constraints (SDC). The design constraints are defined in an SDC file, and timing analysis is performed to verify that the design meets timing requirements.

## Tools Used
- OpenSTA
- Verilog HDL
- SDC
- VS Code

## Project Files
- `design.v` – Verilog design
- `constraints.sdc` – SDC constraints file
- `sta.tcl` – OpenSTA script (if used)
- Timing reports

## Constraints Applied
- Clock definition
- Input delay
- Output delay
- Clock uncertainty (if applicable)

## Result
Static Timing Analysis was completed successfully, and timing reports were generated to verify the design.
